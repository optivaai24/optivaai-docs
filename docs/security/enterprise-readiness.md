# Security & Enterprise Readiness

OptivaAI platforms are designed to meet enterprise security, governance, and compliance expectations.

## Core Principles
- Customer data ownership remains with the customer
- Role-based access control
- Audit logs for all interactions

## Deployment Models
- Cloud (SaaS)
- Hybrid
- On-premise

## Data Handling
- Data is processed only for agreed use cases
- Retention policies are defined per customer
- No shared model training on customer data without explicit agreement

Detailed security documentation is shared during commercial engagement.
