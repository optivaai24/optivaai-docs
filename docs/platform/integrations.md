# Integrations Overview

OptivaAI platforms are designed to integrate with existing enterprise systems through secure and controlled interfaces.

## Integration Categories
- CRM systems
- Telephony platforms
- Ticketing and case management tools
- Data warehouses and BI platforms

## Integration Principles
- Customer systems remain the system of record
- Access is role-based and auditable
- Integrations are configured per deployment

Specific technical details are shared during implementation.
